📊 Customer Conversion Funnel Analysis (Power BI)
🚀 Project Overview

This project analyzes user behavior across a marketing funnel to understand how customers move from product viewing to final purchase. The goal is to identify drop-offs and improve conversion rates.

📁 Dataset Used

A simplified funnel dataset was created with the following stages:

Visitors
Product Views
Add to Cart
Purchase
📊 Dashboard Features
KPI Cards (Total Users, Total Purchases, Conversion Rate)
Funnel Visualization of user journey
Bar Chart for stage-wise comparison
Insights on drop-off points
🔍 Key Insights
Most users drop off after viewing products.
Only a small percentage complete the purchase stage.
Conversion rate decreases at each funnel stage.
🛠 Tools Used
Power BI Desktop
Data Visualization
Basic DAX Measures
📌 Project Outcome

This project demonstrates how user behavior data can be used to analyze conversion performance and identify improvement opportunities in marketing funnels.
